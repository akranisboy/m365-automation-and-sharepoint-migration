# Microsoft 365 Automation & SharePoint Migration

Projeto baseado em experiência real com automação e migração de ambientes Microsoft 365 utilizando PowerShell.

##  Destaques

- Automação de rotinas administrativas
- Scripts aplicados em ambiente corporativo
- Migração real de SharePoint com grande volume de dados

##  Case Real

- Migração de mais de **2.65 TB de dados**
- Aproximadamente **1 milhão de arquivos**
- Estrutura migrada de subsites para sites independentes
- Processo com validação, delta e cutover

## Projetos

### Exchange Retention Automation
Execução de políticas de retenção e Managed Folder Assistant.

### User Provisioning
Automação de criação de usuários e configuração inicial.

### SharePoint Migration Tool
Script para migração de arquivos entre sites com:
- Download automático
- Recriação de estrutura
- Upload com tratamento de erro
- Logs de execução

## Tecnologias

- PowerShell
- PnP PowerShell
- Microsoft 365
- SharePoint Online
- Exchange Online
